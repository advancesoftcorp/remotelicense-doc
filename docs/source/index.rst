============================================
remoteLicense
============================================

アドバンスソフト製品を使うためのLinuxライセンスサーバーをWindows上から操作できる補助ツールです。

- Linuxマシンでライセンス申請するために必要な情報の取得
- ライセンスファイル・ライセンスサーバー実行ファイルの転送
- 製品の実行に必要なライセンスファイルパスの設定
- ライセンスサーバーの起動・終了・状態確認

が直接コマンドを実行することなくGUI上で行えます。
